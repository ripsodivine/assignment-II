World Wide Web

A global information system of interlinked hypertext documents and multimedia resources accessible over the internet.

The internet is the physical infrastructure, the Web is a service that runs on top of it.

The role of the World Wide Web is to provide a user-friendly interface for accessing and sharing information.

Key Components:

The system relies on three main technologies:

  • HTTP/HTTPs: communication protocols. HTTP is the basic request-response model
  • URLs (unique addresses) to address resources
  • HTML - to structure and present content

Browsers act as clients that request and render these resources.
