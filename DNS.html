DNS (Domain Name System)

DNS is a distributed, hierarchical database that translates human‑friendly domain names (e.g., www.wikipedia.org) into machine‑readable IP addresses (e.g., 208.80.154.224).

It works like a global phonebook. The system involves:

  • Domain Name Servers (DNS Servers): These are the computers that store and manage the DNS records.
  • DNS Records: These are the entries in the DNS database that map domain names to IP addresses.
  • DNS Resolvers: These are the clients that initiate DNS queries and receive responses.

Role:

To provide a way to translate domain names into IP addresses, making it easier for users to access resources on the Internet.
