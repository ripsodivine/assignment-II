HTML (HyperText Markup Language)

HTML is the standard markup language for creating web pages and web applications. It uses elements (tags) to structure content. Each element is enclosed in angle brackets, e.g., <p> for a paragraph. HTML documents are parsed by browsers into a Document Object Model (DOM) – a tree representation of the page. HTML5, the latest version, supports multimedia, graphics, and semantic elements (<article>, <section>, <nav>). HTML is not a programming language (no logic) but a presentation and structure language.

It consists of:

  • Elements: These are the building blocks of HTML, such as headings, paragraphs, links, images, etc.
  • Tags: These are used to define elements and are typically enclosed in angle brackets (e.g., <h1> for a heading).
  • Attributes: These provide additional information about elements (e.g., <img src="image.jpg" alt="Description">).

Role:

To give raw text and media a meaningful, hierarchical structure that browsers can interpret and render, and that assistive technologies (screen readers) can navigate.
